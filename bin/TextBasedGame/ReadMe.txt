TextFile Format for Text Based Adventure Game

ItemData 
ItemID/ItemName/ItemDescription/If collected/RoomId location

RoomData2:
RoomId/RoomName/RoomDescription/has been visited/AdjacentRooms/Item ID in the room

PuzzleData

MonsterData
MonsterID/MonsterName/Monster Description/Monster Health/Monster Attack Name/Monster room location
